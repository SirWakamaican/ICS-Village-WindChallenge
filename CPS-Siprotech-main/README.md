How to run the simulation:

1. In the Siprotec directory run ./build and ./run
2. In another terminal, go to the main directory and run "python sonic.py"
3. In another terminal, go to the SCADA directory and run ./build and ./run
4. In another terminal, go to the Attacker directory and run ./buld and ./run
